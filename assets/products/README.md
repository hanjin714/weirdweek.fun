# 产品图片素材

## 使用方法

把 Temu 上找到的商品截图放到对应的编号文件夹里，命名为 `product.jpg`（或 `.png`）。

| 编号 | 商品 | 放图位置 |
|------|------|----------|
| 01-featured | 会夸人的化妆镜 | `01-featured/product.jpg` |
| 02-socks | 单只独立包装袜子 | `02-socks/product.jpg` |
| 03-umbrella | 盆栽迷你遮阳伞 | `03-umbrella/product.jpg` |
| 04-keyboard | 键盘敲击器 | `04-keyboard/product.jpg` |
| 05-banana | 香蕉弯曲度测量仪 | `05-banana/product.jpg` |
| 06-fish | 手机屏幕清洁鱼 | `06-fish/product.jpg` |
| 07-clock | 逃跑闹钟 | `07-clock/product.jpg` |

## 图片要求

- 格式：JPG 或 PNG
- 建议尺寸：600×600 以上
- 文件大小：< 500KB（加载快）
- 截图来源：Temu 商品页面

## 注意

- 图片不存在时会自动显示 emoji 占位符
- 文件命名建议全部小写，不要中文
- 一期一般 7 款商品，多了可以往上加 08、09...
